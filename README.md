Grad-CAM
---------
>Grad-CAM은 Zhou et al. 이 제안한 Class Activation Mapping(CAM)의 general version 이다. 여기서 CAM은 image classification task에서 관심 영역을 확인 할 수 있도록 식별 가능 지역을 heatmap 형태로 보여주는 딥러닝 시스템 해석 방법 중한가지다.
